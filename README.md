# PassGen
PassGen By JetZux Ltd. Dec 2017

### usage
- Generate Password, 6-16 char : [0-9] [!@#$%^&*] [a-z] [A-Z]

### compiler env
|env   | ver|
| - | - |
|__os__|windows 7 x64|
|__compiler__|MSVC 2015 update3 xp mode|
